Chapter 7: Single-Dimensional Arrays

Opening Problem & Motivation

Problem: Read one hundred numbers, compute their average, and find out how many numbers are above the average.

Challenge: To find numbers above the average, you need to store *all* the numbers first while calculating the sum, *then* calculate the average, and *then* iterate through the stored numbers again to compare them with the average. Storing 100 individual variables (num1, num2, ..., num100) is impractical.

Solution: Use an array, which is a data structure that represents a collection of elements of the same type.

Objectives (Summary from Slide 323)

This chapter covers: declaring array variables, creating arrays, accessing elements using indices, array initializers, common array operations (displaying, summing, finding min/max, shuffling, shifting), the enhanced `for` loop (foreach), copying arrays, passing arrays to methods, returning arrays, variable-length arguments, searching (linear, binary), sorting (selection sort), and using the `java.util.Arrays` class.

Declaring Array Variables

Declares a variable that can hold a reference to an array object.

  • Preferred Syntax: datatype[] arrayRefVar;
    Example: double[] myList;
  • Alternative Syntax: datatype arrayRefVar[];
    Example: double myList[]; (Less preferred)

This only declares the reference variable; it does not create the array itself yet.

Creating Arrays

Uses the `new` keyword to allocate memory for the array object.

Syntax: arrayRefVar = new datatype[arraySize];

  • arraySize specifies the number of elements the array will hold.
  • Example: myList = new double[10]; (Creates an array to hold 10 double values).

Declaring and Creating in One Step

Syntax: datatype[] arrayRefVar = new datatype[arraySize];

Example: double[] myList = new double[10];

Array Size and Default Values

  • Once created, an array's size is fixed.
  • You can get the size (number of elements) using arrayRefVar.length.
    Example: myList.length would return 10 for the array above. Note: length is a data field, not a method, so no parentheses `()` are used.
  • When an array is created, its elements are automatically initialized to default values:
    • 0 for numeric types (byte, short, int, long)
    • 0.0 for float and double
    • '\u0000' (the null character) for char
    • false for boolean
    • null for reference types (like `String` or other objects)

Accessing Array Elements (Indexed Variables)

  • Elements are accessed using their index (position).
  • Indices are 0-based, meaning they start from 0 and go up to arrayRefVar.length - 1.
  • Syntax: arrayRefVar[index]
  • Example: For double[] myList = new double[10];
    • myList[0] accesses the first element.
    • myList[9] accesses the last element.
    • myList[10] would cause an ArrayIndexOutOfBoundsException at runtime.
  • An indexed variable (like `myList[2]`) can be used just like a regular variable of that type: 
    myList[0] = 5.6; // Assign value
double x = myList[0]; // Read value
myList[1] = myList[0] + 2.3;
System.out.println(myList[1]);

Array Initializers

A shorthand way to declare, create, and initialize an array in one statement.

Syntax: datatype[] arrayRefVar = {value0, value1, ..., valuek};

  • The size of the array is implicitly determined by the number of values provided.
  • Example: double[] myList = {1.9, 2.9, 3.4, 3.5}; (Creates an array of size 4).
  • This is equivalent to: 
    double[] myList = new double[4];
myList[0] = 1.9;
myList[1] = 2.9;
myList[2] = 3.4;
myList[3] = 3.5;
Caution: The array initializer syntax (`{...}`) can only be used in the declaration statement. You cannot use it for re-assigning an array variable later. 
// Correct:
double[] myList = {1.0, 2.0};

// WRONG:
double[] anotherList;
// anotherList = {3.0, 4.0}; // Compile Error!
// Correct way to reassign:
anotherList = new double[]{3.0, 4.0}; // Or use loop etc.

Processing Arrays (Common Operations)

Arrays are often processed using loops, especially `for` loops.

1. Initializing with Input Values

java.util.Scanner input = new java.util.Scanner(System.in);
System.out.print("Enter " + myList.length + " values: ");
for (int i = 0; i < myList.length; i++) {
    myList[i] = input.nextDouble();
}

2. Initializing with Random Values

for (int i = 0; i < myList.length; i++) {
    myList[i] = Math.random() * 100; // Random doubles between 0.0 and 100.0
}

3. Printing Arrays

for (int i = 0; i < myList.length; i++) {
    System.out.print(myList[i] + " ");
}
System.out.println(); // Add a newline at the end

4. Summing All Elements

double total = 0;
for (int i = 0; i < myList.length; i++) {
    total += myList[i];
}
System.out.println("Sum is " + total);

5. Finding the Largest Element

double max = myList[0]; // Assume first element is max initially
for (int i = 1; i < myList.length; i++) {
    if (myList[i] > max) {
        max = myList[i]; // Update max if a larger element is found
    }
}
System.out.println("Max element is " + max);

6. Finding the Smallest Index of the Largest Element

double max = myList[0];
int indexOfMax = 0;
for (int i = 1; i < myList.length; i++) {
    if (myList[i] > max) {
        max = myList[i];
        indexOfMax = i; // Update index when max is updated
    }
}
System.out.println("Max element " + max + " found at index " + indexOfMax);

7. Random Shuffling

Rearrange elements randomly. One common algorithm (Fisher-Yates shuffle): Iterate from the last element down to the second. In each iteration `i`, pick a random index `j` between 0 and `i` (inclusive), and swap the elements at `i` and `j`.

for (int i = myList.length - 1; i > 0; i--) {
    // Generate random index j such that 0 <= j <= i
    int j = (int)(Math.random() * (i + 1));

    // Swap myList[i] with myList[j]
    double temp = myList[i];
    myList[i] = myList[j];
    myList[j] = temp;
}

8. Shifting Elements

Example: Shift all elements one position to the left, wrapping the first element around to the last position.

double temp = myList[0]; // Save the first element

// Shift elements left
for (int i = 1; i < myList.length; i++) {
    myList[i - 1] = myList[i];
}

// Move the first element to the last position
myList[myList.length - 1] = temp;

Enhanced `for` Loop (Foreach Loop)

Introduced in JDK 1.5, provides a simpler syntax for iterating through all elements of an array (or other collections) sequentially from beginning to end.

Syntax: for (elementType variable : arrayRefVar) { // process variable }

  • elementType must match the type of elements in the array.
  • variable is a temporary variable that holds the value of the current element in each iteration.
  • arrayRefVar is the array being iterated over.

Example: Printing array elements

double[] myList = {1.9, 2.9, 3.4, 3.5};
for (double value : myList) {
    System.out.print(value + " "); // Prints 1.9 2.9 3.4 3.5
}
System.out.println();

Limitations:

  • Always iterates from the first element to the last. You cannot change the order or skip elements.
  • Provides read-only access to elements. Assigning a new value to the loop variable (`value` in the example) does not change the element in the original array.
  • You don't have access to the element's index within the loop.

Use a traditional `for` loop with an index if you need to modify elements, access indices, or iterate in a non-sequential order.

Opening Problem Revisited (AnalyzeNumbers)

Use an array to store the numbers, calculate the average, and count numbers above the average.

// AnalyzeNumbers.java (Conceptual)
import java.util.Scanner;

public class AnalyzeNumbers {
    public static void main(String[] args) {
        Scanner input = new Scanner(System.in);
        System.out.print("Enter the number of items: ");
        int n = input.nextInt();
        double[] numbers = new double[n]; // Create array of size n
        double sum = 0;

        System.out.print("Enter the numbers: ");
        for (int i = 0; i < n; i++) {
            numbers[i] = input.nextDouble();
            sum += numbers[i];
        }

        double average = sum / n;

        int count = 0; // Count numbers above average
        for (int i = 0; i < n; i++) {
            if (numbers[i] > average) {
                count++;
            }
        }

        System.out.println("Average is " + average);
        System.out.println("Number of elements above the average is " + count);
        input.close();
    }
}

Copying Arrays

Assigning one array reference variable to another (list2 = list1;) does not copy the array content. It makes both variables point to the same array object in memory.

To create a separate copy of the array content, use:

1. Using a Loop

int[] sourceArray = {2, 3, 1, 5, 10};
int[] targetArray = new int[sourceArray.length]; // Must create new array

for (int i = 0; i < sourceArray.length; i++) {
    targetArray[i] = sourceArray[i]; // Copy each element
}

2. Using the `System.arraycopy` Method

A more efficient way to copy arrays.

Syntax: System.arraycopy(sourceArray, srcPos, targetArray, destPos, length);

  • sourceArray: The array to copy from.
  • srcPos: Starting index in the source array.
  • targetArray: The array to copy to (must already exist and be large enough).
  • destPos: Starting index in the destination array.
  • length: The number of elements to copy.
int[] sourceArray = {2, 3, 1, 5, 10};
int[] targetArray = new int[sourceArray.length];

System.arraycopy(sourceArray, 0, targetArray, 0, sourceArray.length);
// Now targetArray contains {2, 3, 1, 5, 10}

3. Using the `clone()` method

Creates a shallow copy of the array.

int[] sourceArray = {2, 3, 1, 5, 10};
int[] targetArray = sourceArray.clone();

4. Using `Arrays.copyOf()` or `Arrays.copyOfRange()`

Convenient methods from `java.util.Arrays`.

import java.util.Arrays;
// ...
int[] sourceArray = {2, 3, 1, 5, 10};
int[] targetArray = Arrays.copyOf(sourceArray, sourceArray.length); // Full copy
int[] partialArray = Arrays.copyOfRange(sourceArray, 1, 4); // Copies index 1, 2, 3 -> {3, 1, 5}

Passing Arrays to Methods

  • When an array is passed to a method, the method receives a copy of the array's reference (pass-by-value, where the value is the reference).
  • This means the method can access and modify the original array's elements.
public static void main(String[] args) {
    int[] list = {3, 1, 2, 6, 4, 2};
    printArray(list); // Pass the reference
}

public static void printArray(int[] array) { // Parameter 'array' gets the reference
    for (int i = 0; i < array.length; i++) {
        System.out.print(array[i] + " ");
    }
    System.out.println();
    // array[0] = 999; // If uncommented, this WOULD change list[0] in main
}

Anonymous Arrays

You can create and pass an array directly without declaring a reference variable, useful for one-time use.

Syntax: new dataType[]{value0, value1, ..., valuek}

// Invoke printArray using an anonymous array
printArray(new int[]{3, 1, 2, 6, 4, 2});

Returning Arrays from Methods

A method can return an array. The return type in the method header should be the array type (e.g., `int[]`).

public static void main(String[] args) {
    int[] list1 = {1, 2, 3, 4, 5, 6};
    int[] list2 = reverse(list1); // list2 gets the reference to the new reversed array
    printArray(list2); // Should print 6 5 4 3 2 1
}

/** Returns a new array that is the reverse of the input list */
public static int[] reverse(int[] list) {
    int[] result = new int[list.length]; // Create a new array for the result

    for (int i = 0, j = result.length - 1; i < list.length; i++, j--) {
        result[j] = list[i]; // Copy elements in reverse order
    }

    return result; // Return the reference to the new array
}

Variable-Length Argument Lists (`varargs`)

Allows a method to accept a variable number of arguments of the same type.

Syntax in method declaration: typeName... parameterName (the three dots are crucial).

  • Inside the method, the `parameterName` acts like an array of type `typeName`.
  • Only one varargs parameter is allowed per method.
  • The varargs parameter must be the last parameter in the method signature.
public class VarArgsDemo {
    public static void main(String[] args) {
        printMax(34, 3, 3, 2, 56.5); // Pass multiple doubles
        printMax(new double[]{1, 2, 3}); // Can also pass an existing array
        printMax(); // Can call with zero arguments for the varargs part
    }

    // Method with variable-length arguments
    public static void printMax(double... numbers) { // 'numbers' is treated as double[]
        if (numbers.length == 0) {
            System.out.println("No arguments passed");
            return;
        }

        double result = numbers[0]; // Initialize max with the first element

        for (int i = 1; i < numbers.length; i++) {
            if (numbers[i] > result) {
                result = numbers[i];
            }
        }
        System.out.println("The max value is " + result);
    }
}

The `java.util.Arrays` Class

Provides static utility methods for common array operations.

  • `sort(array)` / `sort(array, fromIndex, toIndex)` / `parallelSort(array)`: Sorts the array (or part of it) in ascending order. `parallelSort` uses multiple processors for potentially faster sorting on large arrays (Java 8+).
  • `binarySearch(array, key)` / `binarySearch(array, fromIndex, toIndex, key)`: Searches for `key` in a sorted array. Returns the index if found, or `-(insertion point + 1)` if not found.
  • `equals(array1, array2)`: Returns `true` if both arrays have the same length and all corresponding elements are equal.
  • `fill(array, value)` / `fill(array, fromIndex, toIndex, value)`: Assigns `value` to all elements (or a range of elements) in the array.
  • `toString(array)`: Returns a string representation of the array content (e.g., `"[2, 4, 7, 10]"`). Very useful for printing arrays quickly.
  • `copyOf(original, newLength)` / `copyOfRange(original, from, to)`: Creates a new array by copying elements from the original.
import java.util.Arrays;

public class ArraysDemo {
    public static void main(String[] args) {
        double[] numbers = {6.0, 4.4, 1.9, 2.9, 3.4, 3.5};
        System.out.println("Original: " + Arrays.toString(numbers));

        Arrays.sort(numbers); // Sort the whole array
        System.out.println("Sorted:   " + Arrays.toString(numbers));

        int[] list = {2, 4, 7, 10, 11, 45, 50};
        System.out.println("Index of 11: " + Arrays.binarySearch(list, 11)); // Output: 4
        System.out.println("Index of 12: " + Arrays.binarySearch(list, 12)); // Output: -6 (insertion point 5)

        int[] list1 = {2, 4, 7, 10};
        int[] list2 = {2, 4, 7, 10};
        int[] list3 = {4, 2, 7, 10};
        System.out.println("list1 equals list2? " + Arrays.equals(list1, list2)); // true
        System.out.println("list1 equals list3? " + Arrays.equals(list1, list3)); // false

        Arrays.fill(list1, 5); // Fill list1 with 5s
        System.out.println("list1 after fill: " + Arrays.toString(list1)); // [5, 5, 5, 5]
    }
}

Command-Line Arguments

You can pass arguments to a Java program when you run it from the command line. These arguments are passed as an array of strings (`String[]`) to the `main` method.

// Example TestMain.java
public class TestMain {
    public static void main(String[] args) { // 'args' receives command-line arguments
        System.out.println("Number of arguments: " + args.length);
        for (int i = 0; i < args.length; i++) {
            System.out.println("Argument " + i + ": " + args[i]);
        }
    }
}

To run this from the command line and pass arguments:

javac TestMain.java
java TestMain arg0 arg1 arg2 "Argument with spaces" 123

Inside the `main` method:

  • `args.length` would be 5.
  • `args[0]` would be "arg0".
  • `args[1]` would be "arg1".
  • `args[2]` would be "arg2".
  • `args[3]` would be "Argument with spaces".
  • `args[4]` would be "123".

Note that all arguments are received as `String`s. If you need numeric values, you must parse them (e.g., using `Integer.parseInt(args[i])`).

Problem: Calculator

Create a program that takes three command-line arguments: operand1, operator, operand2, and performs the calculation.

// Calculator.java
public class Calculator {
    public static void main(String[] args) {
        // Check number of arguments
        if (args.length != 3) {
            System.out.println("Usage: java Calculator operand1 operator operand2");
            System.exit(1);
        }

        // The result of the operation
        int result = 0;

        // Determine the operator
        try {
            int operand1 = Integer.parseInt(args[0]);
            int operand2 = Integer.parseInt(args[2]);

            switch (args[1].charAt(0)) { // Get operator char
                case '+': result = operand1 + operand2; break;
                case '-': result = operand1 - operand2; break;
                case '.': // Use '.' for multiplication as '*' might be expanded by shell
                case '*': result = operand1 * operand2; break;
                case '/': result = operand1 / operand2; break;
                default:
                     System.out.println("Invalid operator: " + args[1]);
                     System.exit(2);
            }
             // Display result
             System.out.println(args[0] + ' ' + args[1] + ' ' + args[2] + " = " + result);

        } catch (NumberFormatException ex) {
             System.out.println("Error: Operands must be integers.");
             System.exit(3);
        }
    }
}

Example command line usage:

javac Calculator.java
java Calculator 10 + 5   // Output: 10 + 5 = 15
java Calculator 20 . 3   // Output: 20 . 3 = 60 (or use '*' if shell allows)
java Calculator 10 / 2   // Output: 10 / 2 = 5

Chapter 8: Multidimensional Arrays

Motivation

One-dimensional arrays model linear collections. To represent data in a grid, matrix, or table format (like distances between cities), we use two-dimensional (2D) arrays.

This table can be represented in Java as:

double[][] distances = {
  {0, 983, 787, 714, 1375, 967, 1087},
  {983, 0, 214, 1102, 1763, 1723, 1842},
  {787, 214, 0, 888, 1549, 1548, 1627},
  {714, 1102, 888, 0, 661, 781, 810},
  {1375, 1763, 1549, 661, 0, 1426, 1187},
  {967, 1723, 1548, 781, 1426, 0, 239},
  {1087, 1842, 1627, 810, 1187, 239, 0}
};

Objectives (Summary from Slide 410)

This chapter covers representing data with 2D arrays, declaring/creating/accessing 2D arrays, common operations (displaying, summing, finding min/max, shuffling), passing 2D arrays to methods, and introduces general multidimensional arrays.

Declaring and Creating 2D Arrays

  • Declaration: datatype[][] arrayRefVar; (e.g., int[][] matrix;)
  • Creation: arrayRefVar = new datatype[rowCount][columnCount]; (e.g., matrix = new int[10][10];)
  • Declaration and Creation: datatype[][] arrayRefVar = new datatype[rowCount][columnCount];
  • Alternative Syntax: datatype arrayRefVar[][] = ...; (Less preferred)
// Example: Create a 5x5 matrix of integers
int[][] matrix = new int[5][5];

// Accessing element at row 2, column 3 (0-based index)
matrix[2][3] = 15;

// Example: Initialize using loops
for (int row = 0; row < matrix.length; row++) { // matrix.length gives number of rows
    for (int col = 0; col < matrix[row].length; col++) { // matrix[row].length gives columns in that row
        matrix[row][col] = row + col;
    }
}

2D Array Illustration

Think of a 2D array as an array where each element is itself another array (representing a row).

Initializing 2D Arrays with Initializers

Use nested braces `{}`.

int[][] matrix = {
    {1, 2, 3}, // Row 0
    {4, 5, 6}, // Row 1
    {7, 8, 9}  // Row 2
};

This creates a 3x3 array.

Lengths of 2D Arrays

  • arrayRefVar.length gives the number of rows.
  • arrayRefVar[rowIndex].length gives the number of columns in the specified `rowIndex`.

Ragged Arrays

  • Since each row in a 2D array is itself an array, rows can have different lengths.
  • This creates a "ragged" or non-rectangular structure.
int[][] raggedArray = {
  {1, 2, 3, 4, 5},
  {2, 3, 4, 5},
  {3, 4, 5},
  {4, 5},
  {5}
};

// raggedArray.length is 5 (number of rows)
// raggedArray[0].length is 5
// raggedArray[1].length is 4
// raggedArray[2].length is 3
// ... etc.

Processing 2D Arrays

Usually involves nested loops, typically `for` loops.

1. Initializing with Input Values

java.util.Scanner input = new java.util.Scanner(System.in);
System.out.println("Enter " + matrix.length + " rows and " +
                   matrix[0].length + " columns: ");
for (int row = 0; row < matrix.length; row++) {
    for (int col = 0; col < matrix[row].length; col++) { // Use matrix[row].length for ragged arrays
        matrix[row][col] = input.nextInt();
    }
}

2. Initializing with Random Values

for (int row = 0; row < matrix.length; row++) {
    for (int col = 0; col < matrix[row].length; col++) {
        matrix[row][col] = (int)(Math.random() * 100);
    }
}

3. Printing Arrays

for (int row = 0; row < matrix.length; row++) {
    for (int col = 0; col < matrix[row].length; col++) {
        System.out.print(matrix[row][col] + " ");
    }
    System.out.println(); // Newline after each row
}

4. Summing All Elements

int total = 0;
for (int row = 0; row < matrix.length; row++) {
    for (int col = 0; col < matrix[row].length; col++) {
        total += matrix[row][col];
    }
}
System.out.println("Sum of all elements is " + total);

5. Summing Elements by Column

// Assumes a non-ragged array or checks bounds carefully
for (int col = 0; col < matrix[0].length; col++) { // Iterate through columns
    int columnTotal = 0;
    for (int row = 0; row < matrix.length; row++) { // Iterate through rows for this column
         if (col < matrix[row].length) { // Check if column exists for ragged arrays
             columnTotal += matrix[row][col];
         }
    }
    System.out.println("Sum for column " + col + " is " + columnTotal);
}

6. Finding the Row with the Largest Sum

int maxRowSum = Integer.MIN_VALUE; // Initialize with smallest possible int
int indexOfMaxRow = -1;

for (int row = 0; row < matrix.length; row++) {
    int currentRowSum = 0;
    for (int col = 0; col < matrix[row].length; col++) {
        currentRowSum += matrix[row][col];
    }
    if (currentRowSum > maxRowSum) {
        maxRowSum = currentRowSum;
        indexOfMaxRow = row;
    }
}
System.out.println("Row " + indexOfMaxRow + " has the largest sum: " + maxRowSum);

7. Random Shuffling (2D)

// Shuffle elements within the 2D array
for (int i = 0; i < matrix.length; i++) {
    for (int j = 0; j < matrix[i].length; j++) {
        int i1 = (int)(Math.random() * matrix.length); // Random row
        int j1 = (int)(Math.random() * matrix[i1].length); // Random col in the random row

        // Swap matrix[i][j] with matrix[i1][j1]
        int temp = matrix[i][j];
        matrix[i][j] = matrix[i1][j1];
        matrix[i1][j1] = temp;
    }
}

Passing 2D Arrays to Methods

Similar to 1D arrays, 2D arrays are passed by reference value. The method receives a copy of the reference and can modify the original array's contents.

public static void main(String[] args) {
    int[][] m = {{1, 2}, {3, 4}, {5, 6}};
    System.out.println("Sum is " + sum(m)); // Pass the 2D array
}

public static int sum(int[][] matrix) { // Method accepts a 2D int array
    int total = 0;
    for (int row = 0; row < matrix.length; row++) {
        for (int col = 0; col < matrix[row].length; col++) {
            total += matrix[row][col];
        }
    }
    return total;
}

Multidimensional Arrays (Beyond 2D)

Java supports arrays with more than two dimensions (n-dimensional arrays).

Declaration: datatype[][][]...[] arrayRefVar;

Creation: new datatype[size1][size2]...[sizeN];

Example: 3D Array

Imagine scores for multiple students, across multiple exams, with each exam having multiple parts (e.g., multiple-choice and essay).

// [student][exam][part]
double[][][] scores = new double[7][5][2]; // 7 students, 5 exams, 2 parts per exam

// Access score for student 3, exam 1, part 0 (multiple-choice)
scores[3][1][0] = 18.5;

Problem: Calculating Total Scores (3D Array)

Calculate the total score for each student across all exams and parts.

// Assume scores[][][] is filled as described
for (int i = 0; i < scores.length; i++) { // Iterate through students
    double totalScoreForStudent = 0;
    for (int j = 0; j < scores[i].length; j++) { // Iterate through exams
        for (int k = 0; k < scores[i][j].length; k++) { // Iterate through parts
            totalScoreForStudent += scores[i][j][k];
        }
    }
    System.out.println("Total score for student " + i + " is " + totalScoreForStudent);
}

Problem: Weather Information (4D Array Example Concept)

Store temperature and humidity for 10 days, 24 hours per day. A 3D array could work: `data[day][hour][measurementType]` where measurementType 0=temp, 1=humidity. Or potentially a 4D array if storing more complex data per hour.

Task: Calculate average daily temperature and humidity.

// Assuming data stored in double[10][24][2] - data[day][hour][0=temp, 1=humidity]
final int NUMBER_OF_DAYS = 10;
final int NUMBER_OF_HOURS = 24;
double[][][] data = new double[NUMBER_OF_DAYS][NUMBER_OF_HOURS][2];

// Assume 'data' is filled by reading the file weather.txt...

System.out.println("Average Daily Temperature and Humidity:");
for (int i = 0; i < NUMBER_OF_DAYS; i++) {
    double dailyTempTotal = 0;
    double dailyHumidityTotal = 0;
    for (int j = 0; j < NUMBER_OF_HOURS; j++) {
        dailyTempTotal += data[i][j][0];
        dailyHumidityTotal += data[i][j][1];
    }
    // Calculate averages
    double avgDailyTemp = dailyTempTotal / NUMBER_OF_HOURS;
    double avgDailyHumidity = dailyHumidityTotal / NUMBER_OF_HOURS;
    System.out.printf("Day %d: Avg Temp = %.2f, Avg Humidity = %.2f%n",
                      (i + 1), avgDailyTemp, avgDailyHumidity);
}
